An honest 0–100, not keyword bingo

  1. Only what's on the page.Claude judges from what's actually written on your resume — it never assumes skills you didn't list.
  2. Must-haves vs nice-to-haves.Every requirement is marked met or missing. Qualified means every must-have is met; the score says how comfortably.
  3. The verdict, with receipts.What matched, what's missing, and why — and the missing list feeds straight into your roadmap.

Run it locally

jobpilot is a Next.js app with a local store — your resume, your keys, and your data never leave your machine.

# 1. Grab it
git clone https://github.com/krishaanth5831/jobpilot.git
cd jobpilot && npm install

# 2. Start it
npm run dev

# 3. Open http://localhost:3000/settings and paste your keys

You'll need a Claude API key (required) and free keys from Adzuna and RapidAPI/JSearch for the widest job coverage — all pasted straight into the Settings page.
